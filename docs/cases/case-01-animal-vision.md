# 案例 01：动物视觉单元

## 1. 基本信息

- 类型：低年级科学探究单元
- 核心主题：不同动物的眼睛结构如何帮助它们生存
- 课时结构：2 课时
- 典型特点：北极星目标清楚，儿童驱动问题明确，体验活动与学具强绑定

## 2. 为什么入库

这个案例值得入库，不是因为它“写得好看”，而是因为它在几个关键层面都做得比较扎实：

- 先有稳定的可持续理解目标，再往下转成孩子能进入的问题
- 每个探究活动都不是孤立体验，而是服务一个明确的生存难题
- 学具不是额外装饰，而是帮助学生切身体验“看到的世界为何不同”
- 最终展示任务能把体验、概念和表达收束成可观察证据

## 3. 案例中的强信号

### 3.1 北极星目标与儿童问题之间有清楚转译

案例先提出教师设计层的核心理解：

- 动物的身体结构是为了帮助它们更好地生存
- 眼睛像为不同动物量身定制的“超级相机”

然后把它转译成低年级孩子能抓住的驱动问题：

- 猫头鹰如何成为“黑夜猎人”
- 马儿怎样做到“眼观六路”
- 蜜蜂如何发现花朵的“秘密通道”

这里的关键不是这三个问题本身，而是“先有稳定理解，再做儿童可理解转译”的动作。

### 3.2 每个活动都围绕“结构 -> 功能 -> 生存问题”推进

案例没有把知识点拆成孤立常识，而是让学生持续回答一个更有意义的问题：

- 这种眼睛结构是什么
- 它带来了什么看到世界的方式
- 这种看到世界的方式解决了什么生存问题

这让活动、概念和生命情境之间形成闭环。

### 3.3 学具直接服务体验与理解

案例中的纸筒、镜片、侧向挡板、隐形“蜜导图”等材料，都不是为了热闹，而是为了模拟感知差异：

- 纸筒帮助理解管状眼的聚焦与转头限制
- 侧向挡板帮助理解马眼的广角视野
- 特殊镜片和图案帮助理解蜜蜂对人眼不可见线索的感知

它们和学习目标是连在一起的。

### 3.4 表现性任务能收束理解

案例最后安排“动物视觉展览会”，让学生用模型、海报或口头介绍解释：

- 特殊结构是什么
- 这让它能怎样看
- 这怎样帮助它更好地生存

这类任务的价值在于，它不是只让学生复述常识，而是要求学生把体验后的理解表达出来。

### 3.5 有差异化和迁移意识

案例还提供了：

- 对不同学生的表达支架
- 课后观察与阅读延伸
- 与环境保护的 STSE 连接
- 从动物视觉迁移到其他动物结构猜测

这说明它不是只追求课堂热闹，而是考虑到理解的迁移与扩展。

## 4. 从案例中抽取出的可迁移原则

这份案例最值得迁移的，不是动物主题，而是以下原则：

1. 先确定北极星理解目标，再把它转成儿童可理解的问题。
2. 每个关键活动都应回答一个真实的小问题，而不是只做体验。
3. 学具必须帮助学生更直接地感受到概念差异。
4. 最终展示任务应能逼出“结构、功能、作用”之间的解释关系。
5. 低年级课程需要表达支架、回顾问题和迁移提问。

## 5. 明确不能照抄的部分

以下内容不应因为本案例有效，就被默认复制到其他课程：

- 不是每个单元都必须拆成 3 个动物或 3 个子问题
- 不是每个课程都必须做工作站轮换
- 不是每个低年级课程都必须采用“超级相机”式比喻
- 不是每个案例都必须以展览会收尾
- 不是每个主题都适合用镜片、眼镜或沉浸式道具

## 6. 对 reviewer / critic 的提醒

如果后续某份课程结果被这个案例影响，应检查的是：

- 是否学到了“转译原则”
- 是否学到了“结构、功能、任务”的扣合
- 是否学到了“体验后再解释”的组织方式

而不是检查：

- 有没有三个驱动问题
- 有没有三个体验站
- 有没有某种相似比喻
- 有没有某种相似展示形式

## 7. 可用于 Rubric 校准的观察点

这个案例可帮助团队校准以下维度：

- 目标-证据-活动一致性
- 驱动问题儿童可理解性
- 学具服务教学目标
- 课时流程现实可执行
- 体验与探究密度
- 差异化与支架支持

## 8. 当前限制

这只是案例库中的第一个正式案例。

在案例数量还少时，不应把它当作默认模板，而应把它当作一个“强信号样本”。
